[[project-name]]
== Project Name

OVSDB Integration

[[features]]
== Features

* REST APIs for OVSDB Server Table Management (CRUD)
* Support for OpenStack Neutron L2 Networks API using OpenDaylight
Mechanism Driver
* Support for VXLAN and GRE Tunnels for overlay mesh networks
